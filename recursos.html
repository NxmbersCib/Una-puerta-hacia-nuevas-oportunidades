¿Qué es la programación?

La programacion se refiere a crear un conjunto de instrucciones que permitan a una computadora o dispositivo llevar a cabo una tarea específica, y a día de hoy se ha convertido en una de las mejores oportunidades laborales que pueden haber, siendo una de los trabajos mejor pagados del siglo, pues en los últimos años se ha convertido en una herramienta indispensable para la creación de nuevas tecnologías.
Existen diversos lenguajes de programación, unos más exigentes que otros, algunos de los ejemplos mas destacados son Java, Python, C++ y JavaScript, sin embargo se hizo enfasis en este ultimo

Visual Studio Code

Visual Studio Code es un editor de código fuente multiplataforma gratuito y desarrollado por Microsoft. El autocompletado de código (IntelliSense), el soporte para varios lenguajes de programación, la integración con Git y la posibilidad de agregar extensiones para personalizar y mejorar la funcionalidad son algunas de sus características. Además, tiene un terminal incorporado que le permite trabajar simultáneamente con varios proyectos. Es el entorno de desarrollo preferido por los desarrolladores.

MDN Web Docs

El sitio web oficial de Mozilla para la documentación de estándares web y proyectos de Mozilla1 es MDN Web Docs, que anteriormente se conocía como Mozilla Developer Network. Los recursos para desarrolladores se encuentran en este sitio, que documentan tecnologías web como CSS, HTML y JavaScript. Además, proporciona documentación, demos y un foro de preguntas y respuestas, donde puede interactuar con otros desarrolladores y plantear preguntas.
Esta fue una fuente principal de información durante todo el proyecto ya que proporciona información veridica sobre las funciones de JavaScript, su estructura y tipos de variables.

JavaScript

JavaScript es un lenguaje de programación interpretado, que forma parte del estándar ECMAScript. Se describe como un lenguaje orientado a objetos, fundamentado en prototipos, de naturaleza imperativa, con tipado débil y dinámico. Se utiliza en aplicaciones con funcionalidad tanto del lado del cliente como del lado del servidor en la web, integrándose en documentos HTML presentados al usuario. De esta manera, se posibilita la creación de aplicaciones con características similares a las CGI, un mecanismo utilizado para generar páginas dinámicas

HTML

El lenguaje fundamental de la World Wide Web es HTML (HyperText Markup Language). Originalmente diseñado como un lenguaje para explicar documentos científicos semánticamente, se ha adaptado para describir la estructura fundamental de páginas web y aplicaciones en línea. Lo cual lo hace un lenguaje fundamental para aprender, principalmente por lo común e importante que son las páginas web para el Internet, por lo que aprenderlo es una habilidad indispensable para un programador.

C++

C++ es un lenguaje de programación que agrega la capacidad de manipulación de objetos al lenguaje C. En los últimos años, se ha convertido en uno de los idiomas más demandados debido a su gran potencia, a pesar de su antigüedad. Es un lenguaje multiparadigma porque combina paradigmas de programación orientada a objetos y estructurada. Debido a esto es muy útil aprender a utilizarla debido a que con este se pueden llegar a hacer proyectos más complejos como crear un juego completo a partir de ello.

Python

Python es un lenguaje de programación interpretado, multiparadigma y multiplataforma. A pesar de haber estado durante décadas, su popularidad sigue creciendo debido a sus características distintivas, como su sintaxis simple y variada en las librerías que puede utilizar, lo que le da mucha flexibilidad. Haciendolo muy útil en situaciones variadas desde desarrollo de software hasta inteligencias artificiales pudiendo hacer cálculos muy pesados de forma más efectiva.
