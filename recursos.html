Visual Studio Code

Visual Studio Code es un editor de código fuente multiplataforma gratuito y desarrollado por Microsoft. El autocompletado de código (IntelliSense), el soporte para varios lenguajes de programación, la integración con Git y la posibilidad de agregar extensiones para personalizar y mejorar la funcionalidad son algunas de sus características. Además, tiene un terminal incorporado que le permite trabajar simultáneamente con varios proyectos. Es el entorno de desarrollo preferido por los desarrolladores.

MDN Web Docs

El sitio web oficial de Mozilla para la documentación de estándares web y proyectos de Mozilla1 es MDN Web Docs, que anteriormente se conocía como Mozilla Developer Network. Los recursos para desarrolladores se encuentran en este sitio, que documentan tecnologías web como CSS, HTML y JavaScript. Además, proporciona documentación, demos y un foro de preguntas y respuestas, donde puede interactuar con otros desarrolladores y plantear preguntas.
Esta fue una fuente principal de información durante todo el proyecto ya que proporciona información veridica sobre las funciones de JavaScript, su estructura y tipos de variables.
